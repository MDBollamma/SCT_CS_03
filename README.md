Password Strength Checker
This is a simple Python script that evaluates the strength of a password based on multiple criteria. It provides a strength rating and helpful suggestions to improve weak passwords.
Features
- Checks password strength based on:
- Minimum length (8 characters)
- Presence of uppercase letters
- Presence of lowercase letters
- Inclusion of digits
- Use of special characters
- Provides a strength rating from "Extremely Weak" to "Very Strong"
- Offers suggestions to improve weak passwords
- Interactive command-line interface
Requirements
- Python 3.x
- No external dependencies (uses built-in re module)
Usage
- Clone the repository or download the script.
- Run the script in your terminal:
python password_checker.py
- Enter a password when prompted.
- View the strength rating and suggestions for improvement.
